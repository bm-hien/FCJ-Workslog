# First Cloud AI Journey 2026 - Internship Worklog

This repository serves as the official documentation and weekly worklog for my internship at the First Cloud AI Journey (FCJ) 2026 program. It records technical learnings, practical implementations, and project milestones throughout the internship.

## Repository Objectives

1.  **Documentation:** To maintain a detailed record of weekly tasks, challenges, and solutions.
2.  **Knowledge Base:** To store technical notes regarding AWS services (EC2, S3, Lambda, Bedrock, etc.) and AI project implementations.
3.  **Project Management:** To track the progress of Team TheBois and individual contributions.

## Structure

The repository is organized by weeks, corresponding to the internship timeline:

* **Week 1:** Onboarding, Environment Setup (AWS CLI, GitHub, Discord), and AWS Exploration Labs.
* **Week 2:** Storage & Databases (S3, EBS, RDS), Hands-on File Upload System implementation, and finalized Project Proposal ("Website Security Baseline Assessment Platform").
* **Week 3:** Networking & Infrastructure Security.

## Technology Stack

This documentation site is built using:

* **Hugo:** Static Site Generator.
* **Markdown:** Content formatting.
* **Git & GitHub:** Version control and hosting.
* **AWS:** The primary cloud platform used for all practical exercises.

## Local Development

To run this project locally on your machine, follow these steps:

1.  Clone the repository:
    git clone https://github.com/FappLord/FCJ-Workslog.git

2.  Navigate to the project directory:
    cd FCJ-Workslog

3.  Start the Hugo server:
    hugo server

4.  Open your browser and navigate to:
    http://localhost:1313
