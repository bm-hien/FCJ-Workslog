### 1. Mục tiêu

* **Khởi động dự án:** Chốt kiến trúc tổng thể và phân công công việc trong nhóm.
* **Thiết lập Backend:** Các thành viên phụ trách backend khởi tạo server, database và hệ thống xác thực.
* **Khởi động Frontend:** Bắt đầu xây dựng giao diện web — landing page, trang đăng nhập/đăng ký.
* **Bắt đầu phát triển:** Dự án GuardScript chính thức bắt đầu ngày **06/03/2026**.

### 2. Chi tiết công việc trong tuần

| Thứ | Công việc chính | Chi tiết | Trạng thái |
|:---:|:---|:---|:---:|
| **Hai** | **Rà soát kiến trúc** | - Chốt kiến trúc tổng thể với team.<br>- Phân công: bản thân phụ trách frontend, các thành viên phụ trách backend và infrastructure. | Hoàn thành |
| **Ba – Tư** | **Thiết lập Backend (Nhóm)** | - Các thành viên khởi tạo server Express.js, cơ sở dữ liệu SQLite, hệ thống đăng nhập.<br>- Hỗ trợ review code và góp ý kiến trúc. | Hoàn thành |
| **Năm** | **Bắt đầu Frontend** | - Bắt đầu xây dựng giao diện landing page và các trang đăng nhập/đăng ký.<br>- Hỗ trợ kiểm tra API backend cơ bản. | Hoàn thành |
| **Sáu** | **Cải thiện UI & Dọn dẹp** | - Cải thiện giao diện — chỉnh sửa spacing, colors, responsive.<br>- Cấu hình `.gitignore`, dọn dẹp file không cần thiết trong repo. | Hoàn thành |

### 3. Kết quả đạt được

#### Backend (Đóng góp nhóm):
* Server backend và cơ sở dữ liệu đã được thiết lập.
* Hệ thống đăng nhập/đăng ký hoạt động.

#### Frontend (Đóng góp cá nhân):
* Bắt đầu xây dựng landing page, trang login/register.
* Thiết lập cấu trúc thư mục frontend và styles cơ bản.

### 4. Vấn đề & Giải pháp
* **Vấn đề:** Cần phối hợp giữa frontend và backend để đảm bảo API hoạt động đúng.
* **Giải pháp:** Trao đổi thường xuyên với team, review code lẫn nhau.

### 5. Bước tiếp theo
* Tiếp tục phát triển frontend dashboard và workspace.
* Nhóm hoàn thiện các API còn lại.
* Xây dựng module mã hóa.
